7. 指标计算与配置比例生成

基于第6步提取的年度财报数据、第4步的CPI数据、第5步的收盘价数据，以及当期的实际十年期国债收益率（通过中债估值或宏观数据获取），执行以下计算流程，最终生成逐年的配置比例文件：

7.1 个股层面计算（遍历每只成分股）

对每只成分股 i 执行：
- 通胀调整盈利：读取最近10个财年的归母净利润 E{i,t} 和对应年度CPI，计算实际盈利 E{i,t}^{real} = E{i,t} imes \frac{CPI{current}}{CPI_t}
- 十年平均盈利：\bar{E}i = \frac{1}{10}\sum{t=-9}^{0} E{i,t}^{real}
- 当前市值：MarketCap_i = ext{收盘价}i imes ext{期末总股本}i
- 自由现金流：FCF_i = ext{经营活动现金流净额}i - ext{资本性支出}i

7.2 指数层面聚合（总和法）

计算沪深300整体指标：
- 指数总市值：MarketCap{index} = \sum MarketCap_i
- 指数CAPE：CAPE{index} = \frac{MarketCap{index}}{\sum \bar{E}i}（总市值除以各成分股十年平均盈利之和）
- 指数FCF Yield：FCF\ Yield{index} = \frac{\sum FCF_i}{MarketCap{index}} imes 100\%

7.3 百分位数转换

获取当年实际十年期国债收益率 r{real}（若无则用名义收益率减2%估算）：
- 危险阈值：U{CAPE} = \frac{1}{r{real} + 0.02}
- CAPE百分位数：
  - 若 CAPE{index} \leq 4.78，则 P{CAPE} = 100
  - 若 CAPE{index} \geq U{CAPE}，则 P{CAPE} = 0
  - 否则 P{CAPE} = \frac{U{CAPE} - CAPE{index}}{U{CAPE} - 4.78} imes 100
- FCF百分位数：
  - 若 FCF\ Yield{index} \leq 3\% 或 \geq 8\%，则 P{FCF} = 0
  - 否则 P{FCF} = \frac{FCF\ Yield{index} - 3\%}{5\%} imes 100

7.4 综合配置系数计算与保存

W{year} = \frac{P{CAPE} + P{FCF}}{2}

保存为 `{年份}_配置比例.csv`，包含字段：
- `年份`（如2005）
- `沪深300CAPE`（数值）
- `沪深300FCFYield`（百分比）
- `实际利率`（用于计算的小数形式，如0.015）
- `CAPE危险阈值`（数值）
- `CAPE百分位`（0-100）
- `FCF百分位`（0-100）
- `权益配置比例W`（0-100，即投入股市的百分比）

8. 策略回测模拟（2005年起，每年投入1万元）

基于第7步生成的逐年配置比例，对四种策略进行资金流入模拟，对比分析2005年至最新年份的长期表现。

8.1 数据准备

获取以下历史收益率数据（2005年至最新年份）：
- 沪深300全收益指数（代码：H00300.CSI，含分红再投资）
- 中债-国债总财富指数（代码：CBA00601.CS，代表国债投资）
- 黄金现货价格（AU9999或伦敦金人民币计价，用于永久组合）
- 月度收益率数据（用于精确模拟月定投，如无法获取则使用年度数据近似）

8.2 策略定义与模拟规则

初始条件：2005年初始资产为0，每年新增可投资资金10,000元（含税和费率假设已包含在净值中）。

策略A：双因子动态配置（目标策略）
- 资金投入：每年首个交易日（或4月30日，与财报数据匹配）投入新增10,000元，按当年 W{year}\% 买入沪深300ETF，(100-W{year})\% 买入国债ETF
- 存量增长：原有持仓按当年各资产收益率增长
- 再平衡：年度再平衡至目标比例 W{year}（卖出/买入使权益仓位等于当年新计算的W）

策略B：年度定投沪深300
- 资金投入：每年首个交易日投入10,000元全仓买入沪深300ETF
- 存量增长：全部资产按沪深300年度收益率增长
- 特点：始终100%权益仓位，无债券配置

策略C：月度定投沪深300
- 资金投入：每月首个交易日投入 10,000/12 \approx 833.33 元买入沪深300ETF
- 存量增长：按月度收益率逐月复利计算（若只有年度数据，则近似为年中投入，当年收益按50%权重计算）
- 特点：平滑年度波动，平均成本效应

策略D：永久投资组合
- 配置比例：固定25%沪深300 + 25%国债 + 25%黄金 + 25%现金（或货币基金）
- 资金投入：每年新增10,000元按上述比例分配
- 再平衡：每年末强制再平衡至25/25/25/25（卖出超出部分，买入不足部分）

8.3 模拟计算流程

对每个年份从2005到2024：
1. 读取当年各资产收益率（股/债/金）
2. 读取当年目标配置比例 W（仅策略A使用）
3. 对于每个策略：
   - 计算新增资金分配
   - 计算年末总资产 = (年初总资产 + 新增资金) × 组合收益率（对于定投需逐笔计算时间权重）
   - 记录年末总资产、累计投入本金、累计收益率

8.4 结果计算与保存

计算各策略的绩效指标并保存为 `回测结果对比.csv` 和 `策略表现分析报告.md`：

核心指标计算：
- 累计收益率：(期末总资产 - 累计投入本金) / 累计投入本金 × 100%
- 年化收益率：(期末总资产 / 累计投入本金)^{(1/年数)} - 1
- 最大回撤：计算历年总资产峰值到谷底的最大跌幅百分比
- 夏普比率：(年化收益率 - 无风险利率) / 年化波动率（使用国债收益率作为无风险利率基准）
- 收益风险比：累计收益率 / 最大回撤

输出文件：
1. `{年份}_策略资产明细.csv`（逐年记录各策略的总资产、本金、收益率）
2. `回测结果对比.csv`（包含各策略的累计收益、年化收益、最大回撤、夏普比率等）
3. `累计净值走势.png`（折线图展示各策略总资产随时间变化）
4. `回撤对比.png`（折线图展示各策略从前期高点的回撤幅度）

8.5 关键观察点（需在报告中体现）
- 双因子策略是否在2007年、2015年、2021年等CAPE高位时降低仓位，规避大跌
- 双因子策略是否在2008年、2018年、2022年等CAPE低位时提高仓位，捕捉反弹
- 与永久组合的波动率对比（目标：更低回撤，相近或更高的长期收益）
